# Contact Book in Go

## Description

A simple Contact Book application built using Go. This project demonstrates basic Go programming concepts such as functions, maps, loops, switch statements, and user input.

## Features

* Add Contacts
* Search Contacts
* View All Contacts
* Delete Contacts
* Menu-Driven Interface
* Exit Option

## Concepts Used

* Variables
* Functions
* Maps
* For Loops
* Switch Statements
* If/Else Conditions
* User Input 

## How to Run

1. Open a terminal in the project folder.
2. Run the following command:

go run switch.go


## Menu

```text
1. Add Contact
2. Search Contact
3. View Contacts
4. Delete Contact
5. Exit
```
